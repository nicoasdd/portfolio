Self-hosted fonts directory.

To self-host fonts, place subset .woff2 files here and reference them via @font-face
declarations in src/styles/global.css.

This implementation uses the system font stack by default (Inter falls back to
system-ui), so no font files are required out of the box. Add a subset Inter
woff2 here if you want a consistent typographic identity offline.

Recommended sources:
  - Google Fonts (download Inter, subset to latin + latin-ext)
  - Fontsource: https://fontsource.org/fonts/inter

Each font file MUST include its license alongside this file.
